# NVIM INSTALLATION

### Windows

*init.lua*
```
mkdir -p $HOME/AppData/Local/nvim
cd $HOME/AppData/Local/nvim
curl -o init.lua https://raw.githubusercontent.com/archways404/dotfiles/main/nvim/init.lua
```

#

### MacOS

*init.lua*
```
mkdir -p ~/.config/nvim
cd ~/.config/nvim
curl -o init.lua https://raw.githubusercontent.com/archways404/dotfiles/main/nvim/init.lua
```
